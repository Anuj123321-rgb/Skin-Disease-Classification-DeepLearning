# 🧠 Skin Disease Classification using Deep Learning

## 📌 Overview

This project focuses on building a **deep learning-based system** for classifying skin diseases using medical images.
It involves extensive experimentation with multiple architectures, hyperparameter tuning, and data augmentation techniques.

---

## 🚀 Key Features

* Implemented multiple deep learning models (CNN, ResNet)
* Applied **data augmentation** to improve generalization
* Performed **hyperparameter tuning** for optimal performance
* Evaluated models using accuracy, precision, recall, and F1-score
* Continuous experimentation to improve robustness

---

## 🛠️ Tech Stack

* Python
* TensorFlow / PyTorch
* OpenCV
* Scikit-learn
* Matplotlib / Seaborn

---

## Dataset 

https://www.kaggle.com/datasets/riyaelizashaju/isic-skin-disease-image-dataset-labelled

## 📊 Experiments

* Tested multiple architectures (Custom CNN, Transfer Learning - ResNet)
* Compared performance across different configurations
* Applied augmentation techniques (rotation, flipping, scaling)
* Tuned learning rate, batch size, optimizer

---

## 📈 Results

* Achieved competitive performance across multiple models
* Improved accuracy through iterative experimentation
* Generated confusion matrix and classification reports
